# Project Title

Gateway for Firebase notifications


## Getting Started


### Installing
Create a Python 3.6 virtual environment and install the packages:
    
    pip install virtualenv
    virtualenv venv
    source venv/bin/activate
    pip install -r requirements.txt
    pip install -e .
 
```
Run
./app.py
```
